C-172 CESSNA SKYHAWK
Quick Reference Normal Operating Checklist

For use only with an approved Cessna checklist

Before Engine Start

1. Preflight: Complete
2. Fuel Drain Tool: Stow
3. Passenger Briefing: Complete
4. Seats: Adjust and Lock
5. Seat Belt & Harness: On
6. Brakes: Set
7. Avionics Master: Off
8. Circut Breakers: In
9. Fuel Selector: On Both
10. Beacon Light: On


Engine Start

Mixture: Rich
Carburetor Heat: Cold
Prime: 2-6 strokes if cold
Master: On
Throttle: Open 1/4 Inch
Propeller Area: Clear
Ignition: To Start
Engine RPM: 800
Oil Pressure: In Green

Note: Oil pressure should be in
green arc within 30 seconds
summer, 60 seconds winter.



Before Taxi

Flaps: up
Avionics Master: On
NAV lights: (Sunset-Sunrise)ON
Transponder: Select for taxiing
ATIS: Listen and Copy
Brakes: Realease and Test
Commnication: GRD or CTAF


Before TakeOff(RunUp)

Brakes: Set
Flight Instruments: Set
Engine Instruments: In Green
Avionics: Set Properly for Flight
Mixture: Rich below 3000'
Throttle: 1700 RPM
Magnetos: Check Lt and Rt

NOTE:

max drop during mag check is 125
RPM withib 50 RPM of each other


Stuff Carberator Heat: On, Check
operation, then Off
Engine Instruments: In Green
Ammeter: Check
Suction Guage: Green Arc
Throttle: 800-1000 RPM
Throttle friction lock: adjust
Flight Controls: Free & Correct
Doors and Windows: Secure
Transponder: Proper Code
and select ALT

Landing Ligh As Req...Sunset to
Sunrise
Pitot Heat: As Required


At TakeOff

Wing Flaps: 0 degrees normal
10 degrees soft field
Throttle: Set Full Power
Mixture RICH: lean above 3000'
Airspeed: Rotate at 55 knots
Climb Speed: Vy 73 knots
Vx 59 knots


Climb

Throttle: Full Speed
Mixture: Rich, Lean above 3000
Trim: Set for Climb
Airspeed: Vx 59 knots
Vy 73 Knots
Cruise Climb: 80 knots


Cruise

Throttle: 2200-2450 RPM
Trim: Set
Landing Light: Off
Mixture: Lean above 3000'


Descent

ATIS: Listen amd Copy
Altimer: Set
Throttle: Set as required
Carberetor Heat: As required
Trim: As Required
Mixture: Rich as Required
Fuel Selector: On Both
Avionics: Set


Before Landing

Seats,seat belts: Secure
Fuel Selector: Both
Mixture: Rich
CArburetor Heat: On as
Required
Landing Light: As Required


Go Around

Throttle: Full Power
Carburetor Heat: Off
Flaps: IMMEDIATELY 20 degrees
Climb Speed: 55 knots
Flaps up once obstacles clear


Landing

Flaps 10 degrees: Below 110 Knots
Flaps Full: Below 85 Knots
Normal App: 65 Knots
Touchdown: On mainwheels
first, lower nose gently
Braking: Minimum required


After Landing

Flaps: Up
Carburetor Heat: Off
Landing Light: Off
Strobe Lights: Off


Shutdown

Avionics Master: Off
Throttle: 800-1000 RPM
Mixture: Idle Cut-Off
Magnetos: Off-Key on Dash
Master Switch: Off
Control Wheel Lock: Install
Record Hobbs, Tach on invoice
Tie Downs: Install
